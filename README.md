# Music Player

A Music player developed based on clean architecture and multi modular rules.

## Key Technologies:

- Jetpack Compose.
- Coroutines.
- MVVM Architecture.
- Flows
- Media 3
- Datastore
- Room DB
- Retrofit

- ## Features:

- Play music
- Create favorites
- Recently added feature
- search
- Filter songs based on whatever you like
- Find the songs lyrics
- Delete songs
- Create playlists
